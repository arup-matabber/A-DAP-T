\# Vulnerable Support Agent



This is an intentionally vulnerable demo AI-agent project for A-DAP-T.



It contains common AI-agent risk patterns:



\- hardcoded demo API key

\- exposed system prompt

\- direct customer data access

\- refund action without human approval

\- email sending without confirmation

\- missing audit logging

\- weak prompt handling



This folder is used to demonstrate how A-DAP-T detects risky AI-agent design patterns.
